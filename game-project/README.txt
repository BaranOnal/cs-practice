# Flappy Bird (Raylib Version)

This folder contains a simple Flappy Bird clone developed in C using the Raylib library.

## Features
- Gravity-based bird movement
- Pipe generation with randomized gaps
- Collision detection
- Score tracking
- Smooth frame-based animation using Raylib

## Requirements
- Raylib library installed
- C compiler (gcc or clang)

## How to Compile
gcc main.c -o flappy -lraylib -lopengl32 -lgdi32 -lwinmm

## How to Run
./flappy

## Notes
This project is a small game made to practice:
- Basic game loops
- 2D rendering
- Collision logic
- Input handling
- Frame-based updates in Raylib
